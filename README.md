# SceneWeaver · 小说转剧本助手

把小说（3 章以上）自动改编成**结构化、可溯源、可打磨**的剧本（YAML）。

> 七牛云 × XEngineer 暑期实训营 · 第三批次 题三 参赛作品

## 和现有工具有什么不同

现有"小说转剧本"工具（如海马轻帆）多是**黑盒一键出纯文本初稿**，面向微短剧量产。SceneWeaver 定位不同——**给作者的、可持续打磨的结构化改编助手**：

- 🧱 **结构化 YAML 输出**：场 / 动作 / 对白 / 转场都是可程序化编辑的字段，不是一坨纯文本。
- 👥 **跨章人物一致**：全局人物表 + id + 别名，解决"老王 / 王先生 / 他"是同一个人。
- 🔗 **可溯源**：每场戏回指原文章节段落，作者改剧本时能跳回原著对照。
- 🚦 **可信度审阅**：每场带置信度与审阅状态，低置信自动标红、提示重点检查。
- 🧠 **双阶段可解释**：先「理解」（抽人物 / 场景 / 情节）再「视听化改编」（外化内心戏、重写对白）。
